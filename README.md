# Anime Explorer – Android Exam Project

This Android application was developed as part of an exam in Android Programming.

The task was to build an app in Kotlin that fetches and displays anime data from the Jikan API. The application retrieves data from a REST API, parses JSON responses, and presents the information in a structured Android user interface.

## Tech Stack
- Kotlin
- Android Studio
- REST API (Jikan API)

Developed in collaboration with a fellow student.
